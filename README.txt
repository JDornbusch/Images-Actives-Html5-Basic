Images actives basic
====================
Presentation
------------
Images Actives is a GNU GPL software that converts raw pictures into interactive animations with user adde textual and audio content.
For further information, see http://images-actives.crdp-versailles.fr/?lang=en
This jquery plugin is intended to add an html5 export functionnality to Images Actives.
